Web Browser Extensions

Updated - October 2023

This list is the mostly used web extensions in my daily routine. Almost all links lead to the official or GitHub pages, but you are free to search for them in your browser as well.

Check My Links

Check My Links - link checker that crawls through your webpage and looks for broken links.

Clear Cache

Clear Cache - clear your cache and browsing data with a single click of a button.

ColorZilla

ColorZilla - advanced Eyedropper, Color Picker, Gradient Generator and other colorful goodies.

Fake Filler

Fake Filler - form filler that fills all inputs on a page with fake/dummy data.

Fonts Ninja

Fonts Ninja - speed up your design workflow! Identify fonts from any website, bookmark, try, and buy them.

JSON viewer

JSON viewer - the most beautiful and customizable JSON/JSONP highlighter that your eyes have ever seen. It is a Chrome extension for printing JSON and JSONP.

Octotree

Octotree - GitHub on steroids. Browser extension that enhances GitHub code review and exploration.

PerfectPixel

PerfectPixel - allows developers and markup designers put a semi-transparent image overlay over the top of the developed HTML and perform pixel perfect comparison between them.

Talend API Tester

Talend API Tester - Visually interact with REST, SOAP and HTTP APIs.

Wappalyzer

Wappalyzer - technology profiler that shows you what websites are built with.

WAVE Evaluation Tool

WAVE Evaluation Tool - web accessibility evaluation tool developed by WebAIM.org. It provides visual feedback about the accessibility of your web content by injecting icons and indicators into your page.

Web Developer

Web Developer - toolbar button with various web developer tools.

Web Developer Checklist

Web Developer Checklist - analyses any web page for violations of best practices. This extension helps web developers easily discover problem areas in websites.
